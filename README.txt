ENG: Application for accounting expenses/incomes/assets in the household 
     (the repository is based on Java classes). 
RU:  Приложение по учету расходов/доходов/активов в домохозяйстве 
     (хранилище основано на Java классах).

1. MainClass.java 	- the main application          / - основное приложение
2. Expenses.java 	- class for expense accounting  / - класс для учета расходов
3. Income.java		- class for income accounting   / - класс для учета доходов
4. Assets.java 		- class for asset accounting    / - класс для учета активов
5. Details.java   	- class for accounting details  / - класс для учета подробностей
                          about expenses and assets     /   о расходах и активах
